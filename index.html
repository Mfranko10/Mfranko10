COMO CREAR UN PROYECTO EN SILVERSTACK XT

Consta en 5 partes para poder crear un proyecto

PASO 1:

Al abrir el Silverstack podemos editar los parametros del PROJECT OVERVIEW donde se encuentra los datos del personal del producto.

PASO 2:

En este paso se crea los folder o carpetas donde vamos a alojar el proyecto y sus reportes.

  • SILVERSTACK

    En el programa vamos a crear los folder basicos, como lo son:

    • LIBRARY

      • EL CARTEL

        • EC_SEMANA_00

          • EC_DAY001_220426

            • OCF

              • EC_DAY001_220426

                • CAM_A

                • CAM_B

            • WAV

    EC = EL CARTEL(ABREVIATURA)

    220426 = YY/MM/DD

    OCF = Almacenamiento de video

    WAV = Almacenamiento de audio

  • MEMORIAS DE COPIADO (DISCO DE TRAFICO / PEGASUS / THUNDERBAY FLEX 8)

    En las memorias vamos a guardar mas informacion que en el programa, como lo son los reportes, el proyecto,fotogramas, etc.

    • LIBRARY

      • EL CARTEL

        • EC_SEMANA_00

          • EC_DAY001_220426

            • OCF

              • EC_DAY001_220426

                • CAM_A

                • CAM_B

            • WAV

            • LUT

            • DAYLYS

            • STILLS

            • SILVERSTACK_PROJECT

            • REPORT

    EC = EL CARTEL (Abreviatura)

    220426 = YY/MM/DD

    OCF = Almacenamiento de video.

    WAV = Almacenamiento de audio.

    LUT = La gama de color o colorizacion del proyecto.

    DAYLYS = Informacion basica del dia.

    STILLS = Son los fotogramas o imagenes que podemos ven en los reportes

    SILCERSTACK_PROJECT = Donde guardamos o alojamos la copia o el Backup del proyecto.

    REPORT = Van alojados los reportes del dia, se realizan dos reportes por dia. Shooting Day, Clips.
